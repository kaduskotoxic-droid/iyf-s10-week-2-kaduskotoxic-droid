# Website 1

Common HTML tags used on a basic webpage include:

<html> – Root element

<head> – Contains metadata

<title> – Page title

<body> – Page content

<h1> to <h6> – Headings

<p> – Paragraphs

<a> – Links

<img> – Images

<div> – Container

<ul>, <ol>, <li> – Lists

# Website 2

  text-decoration: underline;
